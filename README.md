Resume Website (HTML)

This project is a basic resume website built using pure HTML.
It is designed to practice clean structure, readability, and proper HTML tags without using CSS or JavaScript.

Technologies Used

HTML5

Project Features

Well-structured resume layout

Uses semantic HTML tags like header, section, and footer

Skills and project details are shown using lists

Experience and education are displayed using tables

Easy to read and beginner-friendly code

How to Use

Download or clone the project

Open index.html in any web browser

View the resume webpage

Project Purpose

This project helps beginners understand:

HTML page structure

Proper use of headings (h1 to h3)

Lists and tables in real-world layouts

How to build a resume using only HTML
